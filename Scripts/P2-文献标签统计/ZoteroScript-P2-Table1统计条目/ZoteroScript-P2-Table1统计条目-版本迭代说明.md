## Table1- 条目表格 版本迭代说明

### 概述
Table1- 条目表格模块用于将 Zotero 中的文献条目及其标签信息导出为 Markdown 表格格式。该模块支持多种输出模式，可根据需要选择不同的列组合，包括变量标签（A1-DV 至 A6-CV）、主题标签（Item）、理论标签（Theory）、结果标签（Result）、样本标签（Sample1-3）、方法标签（sMeth）、条目信息标签（V1-def 至 V5-sor）等。表格支持文献引用格式，并可与辅助视图集成，使文献可点击查看。


### 版本号体系说明
条目表格脚本采用统一的版本号体系，便于理解版本关系和功能特性：

#### 版本号格式：`V[major]`
- **V[major]（主版本号）**：表示核心功能版本，决定变量标签前缀（旧版 IV/DV/MV 或新版 A1-DV/A2-IV 等）、基础列格式、输出模式数量和条目信息列数
  - V1：使用旧版变量标签前缀，基础列格式（序号、第一作者、年份、期刊），6 种输出模式
  - V2：使用新版变量标签前缀，基础列格式（文献、年份、期刊），集成 items2documents 和辅助视图，7 种输出模式（新增模式 7），支持条目信息 4 列
  - V3：基于 V2，条目信息从 4 列扩展到 5 列（新增 V5-sor）
  - V4：基于 V3，方法标签从 1 列拆分到 2 列（Method1 和 MethodOther）

**版本对应关系：**  

| 版本 | 创建日期 | 变量标签前缀 | 基础列格式 | 输出模式数 | 条目信息列数 | 方法标签列数 | 主要特性 |
|------|---------|------------|-----------|----------|------------|------------|---------|
| V1 | 2025-10-13 4:17pm | 旧版（IV/, DV/, MV/, Mod/, Med/, CV/） | 序号、第一作者、年份、期刊 | 6 种 | 0 列 | 1 列 | 基础版本 |
| V2 | 2025-10-15 12:48pm | 新版（A1-DV/, A2-IV/, A3-MO/, A4-ME/, A5-INV/, A6-CV/） | 文献（citation 格式）、年份、期刊 | 7 种（新增模式 7） | 4 列（V1-def, V2-mea, V3-con, V4-fut） | 1 列 | 标签前缀更新，集成 items2documents 和辅助视图 |
| V3 | 2025-10-30 3:42am | 新版（同 V2） | 文献（citation 格式）、年份、期刊 | 7 种 | 5 列（新增 V5-sor） | 1 列 | 条目信息扩展至 5 列 |
| V4 | 2025-10-30 4:03am | 新版（同 V2） | 文献（citation 格式）、年份、期刊 | 7 种 | 5 列（同 V3） | 2 列（Method1, MethodOther） | 方法标签拆分至 2 列 |

### V 系列（标准版本）迭代历史

#### V1 - 初始版本（2025-10-13）
**核心特性：**
- 基础的条目表格导出功能
- 使用旧版变量标签前缀（IV/, DV/, MV/, Mod/, Med/, CV/）
- 基础列格式：序号、第一作者、年份、期刊
- 6 种输出模式（1-6）
- 不集成 items2documents 和辅助视图
- 不支持条目信息列（V1-def, V2-mea, V3-con, V4-fut 等）

**变量标签前缀（旧版）：**
- `IV/`：自变量
- `DV/`：因变量
- `MV/`：调节变量
- `Mod/`：调节变量（另一种写法）
- `Med/`：中介变量
- `CV/`：控制变量

**基础列格式：**
- 序号：1, 2, 3, ...
- 第一作者：从 creators 字段提取完整姓名（firstName + lastName）
- 年份：从 date 字段提取四位数字年份
- 期刊：从 publicationTitle 字段提取期刊全称

**输出模式（6 种）：**
1. **模式 1**：变量 (6 列) + 理论 (1 列)
   - 基础列 + IV, DV, MV, Mod, Med, CV, Theory

2. **模式 2**：变量 (6 列) + 理论 (1 列) + 样本 (3 列)
   - 模式 1 + Sample1, Sample2, Sample3

3. **模式 3**：变量 (6 列) + 理论 (1 列) + 结果 (1 列)
   - 模式 1 + Result

4. **模式 4**：变量 (6 列) + 理论 (1 列) + 结果 (1 列) + 样本 (3 列)
   - 模式 1 + Result + Sample1, Sample2, Sample3

5. **模式 5**：结果 (1 列) + 样本 (3 列) + 方法 (1 列)
   - 基础列 + Result, Sample1, Sample2, Sample3, Method

6. **模式 6**：全列 (所有 18 列)
   - 基础列 + IV, DV, MV, Mod, Med, CV, OV, Item, Result, Theory, Sample1, Sample2, Sample3, Method

**特点：**
- ✅ 基础功能完整
- ✅ 支持多种输出模式
- ❌ 使用旧版变量标签前缀（与新的标签体系不兼容）
- ❌ 没有集成 items2documents 和辅助视图（文献不可点击）
- ❌ 不支持条目信息列（V1-def, V2-mea, V3-con, V4-fut 等）

---


#### V2 - 标签前缀更新与功能增强（2025-10-15）
**核心改进：**
- 变量标签前缀从旧版更新为新版（A1-DV/, A2-IV/, A3-MO/, A4-ME/, A5-INV/, A6-CV/）
- 基础列格式更新为：文献（citation 格式）、年份、期刊
- 集成 items2documents 获取文档信息（与可视化保持一致）
- 插入到辅助视图中，使文献可点击查看
- 输出模式从 6 种扩展到 7 种（新增模式 7）
- 支持条目信息列（V1-def, V2-mea, V3-con, V4-fut），共 4 列

**变量标签前缀（新版）：**
- `A1-DV/`：因变量（Dependent Variable）
- `A2-IV/`：自变量（Independent Variable）
- `A3-MO/`：调节变量（Moderator）
- `A4-ME/`：中介变量（Mediator）
- `A5-INV/`：工具变量（Instrumental Variable）
- `A6-CV/`：控制变量（Control Variable）

**基础列格式变化：**

```javascript
// V1: 序号、第一作者、年份、期刊
// 基础列：['序号', '第一作者', '年份', '期刊']
// 第一作者：从creators提取完整姓名

// V2: 文献（citation格式）、年份、期刊
// 基础列：['文献', '年份', '期刊']
// 文献：author(year)[number]格式，例如"Smith (2020) [1]"
// 使用items2documents获取文档信息
```

**items2documents 集成：**

```javascript
// 使用 items2documents 获取文档信息（与可视化保持一致）
const docs = Meet.Zotero.items2documents(items);

// 插入到辅助视图中，使文献可点击查看
Meet.Global.views.insertAuxiliary(docs);

// 从 docs 中获取格式化的引用信息
const doc = docs[index];
const lines = doc.pageContent.split('\n');
// 从第二行提取作者和年份信息
// 构建引用格式：author(year)[number]
```

**输出模式（7 种）：**
- 模式 1-6：与 V1 相同，但使用新版变量标签前缀
- **模式 7（新增）**：变量 (6 列) + 理论 (1 列) + 条目信息 (4 列) + 结果 (1 列)
  - 基础列 + A1-DV, A2-IV, A3-MO, A4-ME, A5-INV, A6-CV, Theory, V1- 变量定义, V2- 测量方法, V3- 研究贡献, V4- 研究展望, Result

**条目信息列（4 列）：**
- `V1-def/`：变量定义
- `V2-mea/`：测量方法
- `V3-con/`：研究贡献
- `V4-fut/`：研究展望

**变化对比：**

| 特性 | V1 | V2 |
|------|----|----|
| 变量标签前缀 | 旧版（IV/, DV/, MV/, Mod/, Med/, CV/） | 新版（A1-DV/, A2-IV/, A3-MO/, A4-ME/, A5-INV/, A6-CV/） |
| 基础列格式 | 序号、第一作者、年份、期刊 | 文献（citation 格式）、年份、期刊 |
| items2documents | ❌ 无 | ✅ 有 |
| 辅助视图 | ❌ 无 | ✅ 有（文献可点击） |
| 输出模式数 | 6 种 | 7 种（新增模式 7） |
| 条目信息列 | ❌ 无 | ✅ 有（4 列） |
| 模式 7 | ❌ 无 | ✅ 变量 (6 列) + 理论 (1 列) + 条目信息 (4 列) + 结果 (1 列) |

**特点：**
- ✅ 使用新版变量标签前缀（与新的标签体系兼容）
- ✅ 集成 items2documents 和辅助视图（文献可点击查看）
- ✅ 支持条目信息列（V1-def, V2-mea, V3-con, V4-fut）
- ✅ 新增模式 7，提供更灵活的列组合

#### V3 - 条目信息扩展（2025-10-30）
**核心改进：**
- 条目信息列从 4 列扩展到 5 列（新增 V5-sor）
- 模式 7 更新：条目信息从 4 列扩展到 5 列

**新增条目信息列：**
- `V5-sor/`：样本与数据来源（Source）

**输出模式（7 种）：**
- 模式 1-6：与 V2 相同
- **模式 7（更新）**：变量 (6 列) + 理论 (1 列) + 条目信息 (5 列) + 结果 (1 列)
  - 基础列 + A1-DV, A2-IV, A3-MO, A4-ME, A5-INV, A6-CV, Theory, V1- 变量定义, V2- 测量方法, V3- 研究贡献, V4- 研究展望, **V5- 文献来源**, Result

**条目信息列（5 列）：**
- `V1-def/`：变量定义
- `V2-mea/`：测量方法
- `V3-con/`：研究贡献
- `V4-fut/`：研究展望
- `V5-sor/`：样本与数据来源（新增）

**变化对比：**

| 特性 | V2 | V3 |
|------|----|----|
| 条目信息列数 | 4 列 | 5 列（新增 V5-sor） |
| 模式 7 条目信息 | 4 列（V1-def, V2-mea, V3-con, V4-fut） | 5 列（V1-def, V2-mea, V3-con, V4-fut, V5-sor） |
| 模式 7 描述 | 变量 (6 列) + 理论 (1 列) + 条目信息 (4 列) + 结果 (1 列) | 变量 (6 列) + 理论 (1 列) + 条目信息 (5 列) + 结果 (1 列) |

**特点：**
- ✅ 条目信息扩展至 5 列（包含数据来源信息）
- ✅ 模式 7 更新，提供更完整的条目信息

#### V4 - 方法标签拆分（2025-10-30）
**核心改进：**
- 方法标签从 1 列拆分为 2 列（Method1 和 MethodOther）
- 模式 5 更新：方法列从 1 列扩展为 2 列
- 模式 6 更新：方法列从 1 列扩展为 2 列，全列从 19 列扩展到 22 列

**方法标签拆分：**

```javascript
// V2/V3: 方法标签（1列）
method: "sMeth/"

// V4: 方法标签（2列）
method1: "sMeth/1",      // 方法1单独一列
methodOther: "sMeth/",   // 其他所有方法

// 标签处理逻辑：先检查 sMeth/1（更具体的前缀），再检查其他所有 sMeth/
```

**输出模式（7 种）：**
- 模式 1-4：与 V3 相同
- **模式 5（更新）**：结果 (1 列) + 样本 (3 列) + 方法 (2 列) + 条目信息 (5 列)
  - 基础列 + Result, Sample1, Sample2, Sample3, Method1, MethodOther, V1- 变量定义, V2- 测量方法, V3- 研究贡献, V4- 研究展望, V5- 文献来源
  - 从 9 列扩展到 14 列（基础 3 + 结果 1 + 样本 3 + 方法 2 + 条目信息 5）

- **模式 6（更新）**：全列 (所有 22 列：变量 6+ 主题理论结果 3+ 样本 3+ 方法 2+ 条目信息 5)
  - 基础列 + A1-DV, A2-IV, A3-MO, A4-ME, A5-INV, A6-CV, Item, Theory, Result, Sample1, Sample2, Sample3, Method1, MethodOther, V1- 变量定义, V2- 测量方法, V3- 研究贡献, V4- 研究展望, V5- 文献来源
  - 从 19 列扩展到 22 列（基础 3 + 变量 6 + 主题理论结果 3 + 样本 3 + 方法 2 + 条目信息 5）

- 模式 7：与 V3 相同

**变化对比：**

| 特性 | V3 | V4 |
|------|----|----|
| 方法标签列数 | 1 列（Method） | 2 列（Method1, MethodOther） |
| 方法标签前缀 | `sMeth/` | `sMeth/1` 和 `sMeth/`（拆分） |
| 模式 5 方法列 | 1 列（Method） | 2 列（Method1, MethodOther） |
| 模式 5 总列数 | 9 列 | 14 列（基础 3 + 结果 1 + 样本 3 + 方法 2 + 条目信息 5） |

### 技术亮点

#### 1. 灵活的输出模式
- 支持 7 种输出模式，可根据需要选择不同的列组合
- 从基础的变量 + 理论组合，到全列模式，满足不同分析需求


#### 2. 标签体系兼容
- V2 及以后版本使用新版变量标签前缀（A1-DV/, A2-IV/, A3-MO/, A4-ME/, A5-INV/, A6-CV/）
- 与新的标签体系完全兼容


#### 3. items2documents 集成
- 使用 items2documents 获取文档信息（与可视化保持一致）
- 插入到辅助视图中，使文献可点击查看
- 提高用户体验和可交互性


#### 4. 条目信息支持
- V2 及以后版本支持条目信息列（V1-def, V2-mea, V3-con, V4-fut, V5-sor）
- 提供更完整的文献信息


#### 5. 方法标签拆分
- V4 版本支持方法标签拆分（Method1 和 MethodOther）
- 提供更细粒度的方法分类


#### 6. 引用格式优化
- V2 及以后版本使用 citation 格式（author(year)[number]）
- 与可视化界面编号对应，便于交叉引用


### 注意事项
1. **版本选择：** V1 使用旧版变量标签前缀，与新的标签体系不兼容，不推荐使用
2. **标签前缀：** V2 及以后版本使用新版变量标签前缀（A1-DV/, A2-IV/, A3-MO/, A4-ME/, A5-INV/, A6-CV/）
3. **items2documents：** V2 及以后版本集成 items2documents 和辅助视图，使文献可点击查看
4. **条目信息：** V2 支持条目信息 4 列，V3 及以后版本支持条目信息 5 列（包含数据来源）
5. **方法标签：** V1-V3 版本使用方法标签 1 列，V4 版本使用方法标签 2 列（Method1 和 MethodOther）
6. **输出模式：** V1 支持 6 种输出模式，V2 及以后版本支持 7 种输出模式（新增模式 7）
7. **基础列格式：** V1 使用 " 序号、第一作者、年份、期刊 "，V2 及以后版本使用 " 文献（citation 格式）、年份、期刊 "
8. **列数统计：** 不同版本的同一模式可能有不同的总列数，需要注意区分
